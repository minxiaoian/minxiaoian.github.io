# [《喜欢是风，爱是归途》（温柔治愈）](https://minxiaoian.github.io/)

喜欢一个人，往往从不经意开始。

也许只是某一天傍晚，你们在同一条街上走着，灯光刚亮，空气里有晚餐的味道。他随口说了一句无关紧要的话，你却莫名记住了很久。后来你才明白，有些记忆不是因为重要，而是因为那一刻心动。

喜欢是风。

它来得突然，热烈，毫无道理。你会反复点开聊天框，又假装只是路过；会在朋友圈里多停留几秒，却不敢点一个赞。你以为自己在控制，其实早就被牵着走。

可爱不是。

爱是归途。

是你加班到深夜，走出写字楼，看见有人站在路灯下等你；是吵架之后沉默很久，还是想把外套披到对方肩上；是明明知道彼此都有缺点，却仍然愿意把未来放在同一个方向。

喜欢让人心跳，爱让人安心。

人一生会遇见很多让你开心的人，但能让你安心的人很少。真正的爱，从来不是轰轰烈烈，而是在无数普通的日子里，你忽然发现——

原来回家这两个字，有了具体的人。
